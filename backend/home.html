How it works

From gesture to
words in milliseconds

A 6-layer pipeline processes your hand movements in real time — from raw webcam pixels to spoken words.

01

Webcam Capture

Your browser captures frames via the MediaDevices API and encodes them as JPEG — 10 frames per second sent to the backend.

02

Landmark Extraction

MediaPipe Hands detects 21 3D keypoints on your hand. We apply min-max normalization to produce 63 translation-invariant features.

03

Dual Prediction

Two RandomForest models run simultaneously — one for static gestures (63 features), one for dynamic words (1890 features over 30 frames).

04

Word Builder + TTS

Hold a gesture for 1.5 seconds to commit it to the sentence builder. The Web Speech API speaks each word aloud immediately.

Features

Everything you need

Built for accessibility, speed, and extensibility. Add new gestures in minutes.

⚡

Real-Time Inference

Predictions delivered at 24+ FPS via persistent WebSocket connection — no page reloads, no polling delays.

🧠

Dual ML Models

Static gestures use a 63-feature model. Dynamic word gestures use a 1890-feature sequence model. Best confidence wins.

🔊

Text-to-Speech

Web Speech API speaks every committed gesture aloud. Adjustable speed and pitch. Works offline.

✍️

Word Builder

Hold any gesture 1.5 seconds to add it to a live sentence. Backspace, space, and clear controls included.

📦

Extensible Pipeline

Add any new gesture in 3 steps: drop videos in a folder, run the dataset script, retrain. No code changes needed.

🎯

Confidence Gating

Predictions below 75% confidence are flagged as low-confidence — preventing false positives from reaching the word builder.
